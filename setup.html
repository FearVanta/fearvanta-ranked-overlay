Setup Guide

Up and running
in 5 minutes

Follow these steps and your overlay will be live on stream. No coding required.

1

Download & Extract

Download the ZIP from the Download page and extract it anywhere on your PC. The Desktop works fine.

Tip: Keep the folder name as-is. The installer uses relative paths so moving files around after setup may break things.
2

Add Your Rank Icons

Place your animated .webm rank icon files into the icons folder:

FearVanta-Ranked-Overlay\overlay\assets\icons\

Name them exactly as follows (lowercase, no spaces):

  • bronze.webm
  • silver.webm
  • gold.webm
  • platinum.webm
  • diamond.webm
  • crimson.webm
  • iridescent.webm
  • top250.webm
3

Run the Installer

Right-click install.bat in the root folder and click Run as administrator.

The installer will automatically:

  • Install Node.js if you don't have it
  • Set up the server to auto-start when Windows boots
  • Start the overlay server immediately
Note: If Windows shows a security warning, click "More info" then "Run anyway". This is normal for unsigned batch files.
4

Add Browser Source in OBS

In OBS Studio, add a new Browser Source to your scene:

  • Click + in Sources → choose Browser Source
  • Make sure Local file is unchecked
  • Set the URL to:
http://localhost:8080/fv_overlay.html
  • Set width and height to fit your layout
  • Click OK
5

Add the Control Dock

In OBS Studio, go to View → Docks → Custom Browser Docks and add:

  • Name: ObsidianHUD
  • URL: http://localhost:8080/fv_dock.html

Click Apply. The dock will appear — drag it anywhere in your OBS layout.

Meld Studio: Go to your panels settings and add a new Browser Panel with the same URL.
6

Update Your Overlay

In the ObsidianHUD dock, enter your details and click Update Overlay:

  • Activision Username — your display name
  • SR — your current skill rating
  • Top 250 Position — unlocks at 10,000 SR
  • Layout — Horizontal or Vertical
  • Background Template — choose your style
  • Background Opacity — 0.0 to 1.0

The overlay updates instantly on stream. No restart needed.

Troubleshooting

Overlay shows blank or won't load
Make sure the server is running. Double-click start_overlay.bat in the overlay folder. Then right-click your Browser Source in OBS and click Refresh.
Dock says "Cannot reach overlay.json"
The server isn't running. Double-click start_overlay.bat in the overlay folder to restart it.
Icons not showing / black box
Make sure your .webm files are in the correct folder with the exact lowercase filenames: bronze.webm, silver.webm, gold.webm, platinum.webm, diamond.webm, crimson.webm, iridescent.webm, top250.webm.
Update button says "Could not save"
The server is not running. Make sure you're using the Node.js server (server.js) and not the old Python server. Run start_overlay.bat to restart.
Port 8080 already in use error
Open Task Manager, find node.exe or python.exe and end the task. Then run start_overlay.bat again.
Top 250 Position field is greyed out
This is expected — the field only unlocks when your SR is 10,000 or above, matching how the game works.
